Welcome to trlX's documentation!
================================
trlX is a library for training large language models with reinforcement learning. It uses two algorithms, PPO for online training and ILQL for offline training using a static dataset, via two backends: Huggingface Accelerate and NVIDIA NeMo. Check out the examples for usage instructions.
